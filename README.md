# Interactive Personal Dashboard

A clean, interactive personal dashboard built with HTML, CSS, and vanilla JavaScript. Features a live clock, motivational quotes, a todo list, and a daily goal tracker — all with a smooth light/dark theme toggle.

## Live Demo

Open `index.html` in any browser — no server or installation needed.

## Features

- **Live Clock** — displays current time in HH:MM:SS format, updates every second
- **Motivational Quote** — random quote selected from a curated list on every page load
- **Todo List** — add, complete, and delete tasks with persistent localStorage storage
- **Daily Goal Progress Bar** — manually track daily progress in 10% increments, resets after 100%
- **Light / Dark Theme** — smooth animated toggle between sky blue day mode and deep night mode
- **Animated Background** — floating cloud and pulsing sun/moon that adapts to the current theme

## Technologies Used

- HTML5
- CSS3 (CSS Variables, Flexbox, Animations)
- Vanilla JavaScript (DOM manipulation, localStorage, setInterval)

## What I Learned

- Using CSS custom properties (variables) for dynamic theming
- Persisting data in the browser using localStorage
- DOM manipulation and event handling in JavaScript
- CSS animations with keyframes
- Building a responsive layout without any frameworks
